Accessibility

This site is designed with web accessibility in mind. Standards for this are defined by Web Content Accessibility Guidelines (WCAG2).

  • Access keys
  • Using access keys in different web browsers
  • Text only version

Access Keys

The site uses as sub-set of the defacto standard acccess keys (as defined in this attempt at an accesskeys standard):

  • 1 - Home
  • 3 - Site Map
  • 0 - Help, Accessibility details
In addition the following key(s) is/are active:
  • S - Skip navigation (to contents)

Using access keys in different web browsers

Firefox (Mac OSX)
Hold down the Ctrl & Option key and press the number or letter of the access key.

Safari (Mac OS X)
Hold down the Ctrl key and press the number or letter of the access key.

Text only Version

The current text only pages are generated from existing pages processed by machine to be 'text only'.
This has its limitations, there are known problems, and should probably be considered 'a work in progress'.
